The base Module¶

Data extraction layer base classes.

This module contains the abstract classes for the Data Extraction Layer of an OM monitor parallelization.

class OmDataEventHandler(monitor_parameters)[source]¶

Bases: object

The base class for an OM DataEventHandler.

The event handler deals with event and event sources, and implements functions to initialize the sources, recover events from them, opening and closing event, and investigate the event content.

Parameters

monitor_params (MonitorParams) – an object storing the OM monitor parameters from the configuration file.

abstract initialize_event_source(source, node_pool_size)[source]¶

Initializes the event source.

This function must be called on the collecting node before the event_generator() function is called on the processing nodes, and performs all the operation needed to initialize the event source (announcement of OM, etc.)

Parameters

  • source (str) – a psana-style DataSource string.

  • node_pool_size (int) – the total number of nodes in the OM pool, including all the processing nodes and the collecting node.

Return type

Any

abstract event_generator(source, node_rank, node_pool_size)[source]¶

Retrieves events to process.

This function must be called on each processing node after the initialize_event_source() function has been called on the collecting node. The function is a generator and it returns an iterator over the events that the calling node must process.

Parameters

  • source (str) – a string describing the data source. The exact format of the string depends on the specific Data Recovery Layer currently being used. See the documentation of the relevant ‘initialize_event_source’ function.

  • node_rank (int) – the rank, in the OM pool, of the processing node calling the function.

  • node_pool_size (int) – the total number of nodes in the OM pool, including all the processing nodes and the collecting node.

Yields

Dict[str, Dict[str,Any]] – a dictionary storing the event data.

Return type

Generator[Dict[str, Any], None, None]

extract_data(event)[source]¶

Extracts data from an event.

This function calls in sequence all the Data Extraction functions, passing the event to event as input to each of them. The function then returns the extracted data.

Parameters

data_extraction_funcs (Dict[str, Callable[[Dict[str,Dict[str,Any]]], Any]]) –

a dictionary containing Data Extraction functions to be attached to the class instance being created.

  • Each dictionary value must store a function implementation.

  • The corresponding dictionary key will define the name with which the function will be attached to the class instance.

Returns

a dictionary storing the values returned by the Data Extraction functions.

  • Each dictionary key identifies a function attached to the event.

  • The corresponding dictionary value stores the data returned by the function.

Return type

Dict[str, Any]
